# 設備品質分析儀表板

## 🔗 網站連結

👉 **[https://homejay-eup.github.io/quality-dashboard/quality-dashboard.html](https://homejay-eup.github.io/quality-dashboard/quality-dashboard.html)**

點擊上方連結即可直接開啟，無需安裝任何軟體，不需等待喚醒。

---

## 使用方式

1. 開啟上方網址
2. 上傳設備品質分析 Excel（多期合併版）
3. 自動產生圖表、KPI、篩選及建議

> Excel 檔案僅在您的瀏覽器本機運算，不會上傳至任何伺服器。

---

## 必要工作表

| 工作表名稱 | 說明 |
|-----------|------|
| 車機_彙整總覽 | 車機品質資料（需含「期間」欄） |
| 鏡頭_彙整總覽 | 鏡頭品質資料（需含「期間」欄） |
| 車機_月趨勢 | 車機月趨勢資料（選填） |
| 鏡頭_月趨勢 | 鏡頭月趨勢資料（選填） |

---

## 功能一覽

- 多期間比較（目前期間 vs 對比期間）
- 廠商 → 類型 → ERP品號 階層篩選
- KPI 指標卡（含 delta 比較）
- 上線量 / 回廠量品質占比圓環圖
- 回廠原因 / 處置結果圓餅圖
- 月趨勢 / 廠商趨勢折線圖
- 詳細資料表（含小計 / 總計）
- 下載篩選結果 .xlsx
- 品管建議 & 採購建議（自動生成 / 可編輯 / 可下載）
- 鏡頭室內/室外環境分析
